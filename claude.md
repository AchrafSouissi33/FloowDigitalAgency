# Project Map & State Tracking

- **North Star:** Build "Agency Sync," a local web application that streamlines daily task updates between a Production Manager (PM) and an Account Manager (AM) in a digital agency. It features a "Walkthrough" flashcard UI for rapid task updates and an "Intervention Dashboard" for unblocking tasks.
- **Integrations:** None for Version 1. No external APIs, no Slack, no Shopify.
- **Source of Truth:** A local SQLite database. No authentication or login system. Toggle UI switch for "PM" or "AM" view.
- **Delivery Payload:** A fully functional Next.js (App Router) local web application using Tailwind CSS and Lucide React icons.
